# 📊 E-commerce Data Analysis using SQL

## 🎯 Objective

The goal of this project is to analyze an e-commerce dataset to understand customer behavior, product performance, and revenue trends.

## 📁 Dataset

* Source: Kaggle (Brazilian E-commerce Dataset)
* Tables used: customers, orders, order_items, products, payments

## 🛠️ Tools Used

* SQL (MySQL)

## 🔍 Key Analysis Performed

* Total revenue calculation
* Top customers identification
* Product category performance
* Monthly revenue trends
* Best-selling products

## 📊 Key Insights

* A small group of customers contributes a large portion of revenue.
* Certain product categories generate significantly higher sales.
* Monthly trends show fluctuations, indicating seasonal demand.

## 💡 Business Recommendations

* Focus on retaining top customers through loyalty programs.
* Increase marketing for high-performing categories.
* Improve or remove low-performing products.

## 🚀 Conclusion

This project demonstrates how SQL can be used to extract meaningful insights from raw data and support business decision-making.
